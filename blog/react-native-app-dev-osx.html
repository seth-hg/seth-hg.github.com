React Native开发环境搭建

Feb 20, 2016

1. 用Homebrew安装Node.js

brew install node

2. 安装React Native命令行工具

npm install -g react-native-cli

3. 安装Android SDK

下载Android SDK，推荐从 AndroidDevTools 的镜像下载。解压到任意目录。设置环境变量ANDROID_HOME，编译时需要。

export ANDROID_HOME=$HOME/Android/sdk

4. 创建项目

react-native-cli init HelloWorld

5. 编译和调试

react-native-cli run-android

该命令完成3件事：第一，启动一个web服务器，监听8081端口，React Native生成的App需要从服务器下载js脚本才能运行；第二，编译生成apk；第三，使用adb命令安装app到目标环境进行调试。

调试app至少需要一个目标环境，模拟器或者真实手机都可以。使用模拟器调试需要用SDK Manager图形界面创建一个模拟器并启动。使用手机调试需要安装手机对应的USB驱动，并打开手机的USB调试支持，将手机通过USB数据线连接到电脑。使用下面命令可以检查手机是否已经正确连接。

adb devices -l

6. 运行App

前面步骤完成后，HelloWorld将被安装到目标环境。直接运行会显示一个红色的错误界面，这是因为js没有正确加载，需要先设置服务器地址和端口。按菜单键或者摇动手机，App上会弹出菜单，选择Dev Settings -> Debug server host & port for device，填上开发机的IP地址和端口（8081）.然后选择Reload JS重新加载。
